Private Web Application

Personal Etsy Shop Management Tool

This is a private web application developed and operated by me for my own Etsy shop. It uses the Etsy Open API v3 with OAuth 2.0 to let me view and manage my shop data — including listings, inventory, orders, and sales reports. The application is for my personal use only and is not offered to any third party.

🛍️ What This Application Does

  • View and manage active shop listings
  • Monitor inventory levels and quantities
  • Track orders and fulfillment status
  • Review sales reports and revenue data
  • Update listing details, prices, and tags
  • Renew and manage listing lifetimes

🔒 Privacy & Data Handling

Personal use only This app is built, tested, and used exclusively by me to manage my own Etsy shop.
No third-party sharing API credentials and shop data are never shared with or exposed to any third party.
No external automation tools API credentials are not entered into any external services or automation platforms.
Data stays private All seller and customer data accessed via the API is kept private and secure.

⚙️ API Usage & Compliance

This application is built to comply with the Etsy API Terms of Use in full. Specifically:

  • Uses OAuth 2.0 for all authenticated requests
  • Requests only the scopes required for each operation
  • Respects Etsy's rate limits at all times
  • Application is tested by me only — no public user base
  • No bulk scraping or data harvesting
  • Follows Etsy seller policy for all listing content

🔧 Technical Overview

A lightweight private application connecting to the Etsy Open API v3:

Etsy Open API v3 OAuth 2.0 HTTPS only Python Private / local deployment
