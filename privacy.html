Artelios Shop Manager Private Application

Privacy Policy

Last updated: May 9, 2026

1. Overview

Artelios Shop Manager is a private web application built and operated by a single individual (the developer) to manage their own Etsy shop via the Etsy Open API v3. This application is not a public service and has no users other than its developer.

2. Data Collected

This application accesses the following Etsy shop data via the Etsy Open API v3, using OAuth 2.0 authorization:

— Active and inactive shop listings (titles, descriptions, prices, images, quantities)
— Inventory levels and stock information
— Orders and fulfillment status
— Sales reports and revenue data

No data from website visitors is collected. This page exists solely to describe the application to Etsy for API registration purposes.

3. How Data Is Used

All data accessed through the Etsy API is used exclusively by the developer to view and manage their own Etsy shop. The data is used for no other purpose.

4. Data Sharing

Shop data and API credentials are never shared with any third party. API credentials are not entered into any external services, automation platforms, or tools. No data is sold, rented, or transferred to any other party under any circumstances.

5. Security

API credentials (OAuth 2.0 tokens) are stored securely on the developer's own infrastructure and are not exposed to any external systems. All communication with the Etsy API is conducted over HTTPS.

6. Rate Limits & API Compliance

This application respects Etsy's API rate limits at all times and is operated in full compliance with the Etsy API Terms of Use.

7. Contact

Questions regarding this application or its use of the Etsy API can be directed to hello@artelios.art.
